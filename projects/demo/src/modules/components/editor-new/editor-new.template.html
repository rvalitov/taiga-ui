Rich Text Editor based on TipTap Editor for using with Angular forms. For safety reasons use a sanitizer with this component.

This component is experimental. Use it with caution. Expect breaking changes
  • Use TuiEditorTool to create an array of built-in tools. Pass this array to editor's property tools.
  • Also, you can create your own tool. You can get access to original TipTap editor API via $implicit of template context object . Add it to array passed as the editor's property tools.
